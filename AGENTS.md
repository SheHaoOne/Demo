### When code is added

Once application code is committed, future agents should:
1. Identify the tech stack from project files (e.g., `.csproj`/`.sln` for .NET, `package.json` for Node.js).
2. Install dependencies using the appropriate package manager.
3. Update the VM environment update script accordingly.
4. Re-examine this file and update these instructions.
